CÓMO OBTENER TU TOKEN DE DERIV
💡 Importante: Usá el navegador del celular o computadora — NO la app de Deriv — para poder copiar el token correctamente.
1

Entrá a tu cuenta Deriv

Abrí Chrome o Safari y entrá a app.deriv.com con tu usuario y contraseña.

2

Ir directo a API Token

Hacé clic acá 👉 app.deriv.com/account/api-token

3

Crear el token

En "Token name" escribí DbotsUltra.
Marcá solo estos permisos:
✅ Read   ✅ Trade   ✅ Trading information
❌ NO marques Payments ni Admin
Hacé clic en "Create".

4

Copiar el token

El token aparece en la lista — hacé clic en el ícono de copiar.
📱 En celular: mantené presionado el token → Copiar.

5

Pegarlo en la app

Volvé a DBots Pro Ultra → Tokens Deriv → pegá en el campo correspondiente → GUARDAR.

6

Demo vs Real — ¿Cuál usar?

Demo: Dinero virtual. Usalo para practicar sin riesgo.
Real: Dinero real. Solo usalo cuando ya probaste en Demo y entendés cómo funciona.
